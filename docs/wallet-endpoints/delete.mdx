---
title: "Delete Wallet"
description:
  "Stop tracking a wallet address. For bulk wallet address deletion, please
  email support@invezo.com or message us on Slack."
api: "POST https://api.invezo.com/v1/wallets/tracking/delete"
---

### JSON Body Parameters

<ParamField query="chain" type="string" required>
  platform to query (valid chains: eth, sol)
</ParamField>

<ParamField query="address" type="string" required>
  wallet address (example: 0xd8da6bf26964af9d7eed9e03e53415d37aa96045)
</ParamField>

<ResponseExample>

```json Response
{
    "status": "OK",
    "request_id": "",
    "data": "Successfully deleted wallet."
}
```

</ResponseExample>
