---
title: "ETH Transactions"
description:
  "Get all transactions for a wallet on Ethereum." 
api: "GET https://api.invezo.com/v1/wallets/tracking/ethereum/transactions"
---

### Query Parameters

<ParamField query="address" type="string" required>
  wallet address (example: 0xd8da6bf26964af9d7eed9e03e53415d37aa96045)
</ParamField>

<ParamField query="min" type="int or string" optional>
  start of the aggregate time window as a timestamp (example: 1652486400 or '2022-05-13T18:00:00')
</ParamField>

<ParamField query="max" type="int or string" optional>
  end of the aggregate time window as a timestamp (example: 1675191600 or '2023-01-31T12:00:00')
</ParamField>

<ParamField query="interval" type="string" optional>
  size of the time window (valid intervals: 10m, 1h, 1d, 1w, 1mo)
</ParamField>

<ParamField query="token" type="string" optional>
  asset contract address (example: EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v)
</ParamField>

<ParamField query="sort" type="string" optional>
  sort the results by timestamp. asc will return results in ascending order
  (oldest to newest), desc will return results in descending order (newest to
  oldest). default value is desc
</ParamField>

<ParamField query="limit" type="int" optional>
  limit the number of results returned, default is 5000 and max is 50000
</ParamField>
