---
title: "Holdings"
description: "Get current holdings of a wallet address."
api: "GET https://api.invezo.com/v1/wallets/tracking/holdings"
---

### Query Parameters

<ParamField query="chain" type="string" required>
  platform to query (valid chains: eth, sol)
</ParamField>

<ParamField query="address" type="string" required>
  wallet address (example: 0xd8da6bf26964af9d7eed9e03e53415d37aa96045)
</ParamField>
